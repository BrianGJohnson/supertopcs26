## Client Components - Any component that uses Next.js `Link` via `component={Link}` or other interactive Mantine elements (Button, Anchor, NavLink, Menu, etc) must be a client component. - These files must start with `"use client";` as the first line. - Example files: PublicHeader, SiteFooter, any navigation bars or interactive footers.
